# Services

Capa de acceso a funcionalidades del juego.

## Subdirectorios

- `SaveService/` - Servicio de guardado
- `AudioService/` - Servicio de audio
- `AssetService/` - Servicio de carga de assets
- `WeatherService/` - Servicio de clima
- `InputService/` - Servicio de entrada
- `LocalizationService/` - Servicio de localización

## Convenciones

- Los services son la capa de acceso para la presentación
- Pueden usar engines, pero engines NO usan services
- Mantener interfaces simples y claras

# Entities

Entidades del juego Endless Swell.

## Subdirectorios

- `player/` - Entidad del jugador
- `npc/` - Personajes no jugadores
- `vehicles/` - Vehículos (furgoneta)
- `surfboards/` - Tablas de surf
- `wetsuits/` - Neoprenos
- `wildlife/` - Fauna marina
- `items/` - Objetos del juego
- `props/` - Objetos decorativos

## Convenciones

- Cada entidad debe implementar una interfaz base
- Las entidades son independientes de Phaser
- Usar composición sobre herencia

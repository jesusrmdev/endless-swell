# TimeEngine

Motor de tiempo para Endless Swell.

## Responsabilidades

- Gestionar tiempo del juego
- Controlar ciclos día/noche
- Gestionar estaciones

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `TimeEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

# VehicleEngine

Motor de vehículo para Endless Swell.

## Responsabilidades

- Gestionar la furgoneta del jugador
- Controlar conducción y física del vehículo
- Gestionar combustible

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `VehicleEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

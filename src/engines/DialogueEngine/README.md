# DialogueEngine

Motor de diálogos para Endless Swell.

## Responsabilidades

- Gestionar diálogos con NPCs
- Controlar flujo de conversaciones
- Evaluar condiciones de diálogos

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `DialogueEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

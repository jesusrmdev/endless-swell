# JournalEngine

Motor de diario para Endless Swell.

## Responsabilidades

- Gestionar entradas del diario
- Buscar y filtrar entradas
- Controlar apertura/cierre del diario

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `JournalEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

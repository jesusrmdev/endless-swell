# InventoryEngine

Motor de inventario para Endless Swell.

## Responsabilidades

- Gestionar el inventario del jugador
- Equipar tablas y neoprenos
- Controlar espacio disponible

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `InventoryEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

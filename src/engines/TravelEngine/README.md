# TravelEngine

Motor de viajes para Endless Swell.

## Responsabilidades

- Gestionar viajes entre regiones
- Controlar rutas y transportes
- Desbloquear nuevas zonas

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `TravelEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

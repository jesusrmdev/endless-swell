# SurfEngine

Motor de surf para Endless Swell.

## Responsabilidades

- Gestionar mecánicas de surf
- Calcular puntuaciones
- Gestionar habilidades del jugador

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `SurfEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

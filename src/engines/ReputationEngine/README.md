# ReputationEngine

Motor de reputación para Endless Swell.

## Responsabilidades

- Gestionar reputación con facciones
- Controlar títulos y niveles
- Rastrear experiencia de reputación

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `ReputationEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

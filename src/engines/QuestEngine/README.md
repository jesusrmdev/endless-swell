# QuestEngine

Motor de misiones para Endless Swell.

## Responsabilidades

- Gestionar misiones del jugador
- Controlar objetivos y recompensas
- Verificar prerrequisitos

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `QuestEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

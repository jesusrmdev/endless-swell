# PhotoEngine

Motor de fotos para Endless Swell.

## Responsabilidades

- Tomar fotos del viaje
- Gestionar galería de fotos
- Aplicar filtros

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `PhotoEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

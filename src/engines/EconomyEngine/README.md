# EconomyEngine

Motor de economía para Endless Swell.

## Responsabilidades

- Gestionar dinero del jugador
- Procesar transacciones
- Gestionar tiendas

## Dependencias

- **Ninguna** - Este engine es completamente independiente

## Archivos

- `EconomyEngine.ts` - Implementación principal
- `types.ts` - Tipos de datos
- `interfaces.ts` - Contratos del engine
- `index.ts` - Exportaciones

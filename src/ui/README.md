# UI

Interfaz de usuario de Endless Swell.

## Subdirectorios

- `hud/` - Heads-Up Display (vida, energía, etc.)
- `menus/` - Menús del juego
- `map/` - Mapa del mundo
- `inventory/` - Interfaz de inventario
- `journal/` - Interfaz del diario
- `dialogue/` - Interfaz de diálogos
- `widgets/` - Componentes reutilizables
- `animations/` - Animaciones de UI
- `transitions/` - Transiciones entre pantallas

## Convenciones

- La UI se renderiza en UIScene
- Separar lógica de UI de la presentación
- Usar eventos para comunicación con engines

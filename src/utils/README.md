# Utils

Utilidades compartidas de Endless Swell.

## Uso

Este directorio contiene funciones de utilidad que son utilizadas
por múltiples módulos del juego.

## Convenciones

- Funciones puras sin efectos secundarios
- Documentar cada función con JSDoc
- Incluir tipos de parámetros y retorno
- Agrupar por categoría (matemáticas, fechas, strings, etc.)

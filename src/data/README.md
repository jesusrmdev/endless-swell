# Data

Datos del juego Endless Swell.

## Principio Data Driven

TODO el contenido se define mediante datos, no código.

Para añadir nuevo contenido, basta con crear nuevos archivos de datos
sin modificar el código existente.

## Subdirectorios

- `beaches/` - Datos de playas
- `regions/` - Datos de regiones
- `surfboards/` - Datos de tablas
- `wetsuits/` - Datos de neoprenos
- `vehicles/` - Datos de vehículos
- `npcs/` - Datos de NPCs
- `dialogues/` - Datos de diálogos
- `academies/` - Datos de academias
- `sponsors/` - Datos de patrocinadores
- `economy/` - Datos económicos
- `weather/` - Datos meteorológicos
- `competitions/` - Datos de competiciones
- `achievements/` - Logros
- `localization/` - Textos localizados

## Formato

- Usar JSON para datos estáticos
- Cada archivo define una entidad o grupo de entidades
- Nombrar archivos en minúsculas con guiones

# World

Construcción y gestión del mundo de Endless Swell.

## Subdirectorios

- `maps/` - Mapas del juego
- `regions/` - Regiones geográficas
- `beaches/` - Playas
- `interiors/` - Interiores (tiendas, casas)
- `roads/` - Carreteras
- `campsites/` - Áreas de acampada
- `surfshops/` - Tiendas de surf
- `academies/` - Academias de surf
- `competitions/` - Zonas de competición
- `tilesets` - Tilesets para mapas

## Convenciones

- Los mapas se crean con Tiled
- Cada tileset tiene su propia carpeta
- Los datos de world building se separan de la lógica

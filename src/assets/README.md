# Assets

Recursos visuales y audio de Endless Swell.

## Subdirectorios

- `sprites/` - Spritesheets y sprites individuales
- `tilesets/` - Tilesets para mapas
- `portraits/` - Retratos de NPCs
- `ui/` - Elementos de interfaz
- `audio/`
  - `music/` - Música de fondo
  - `ambience/` - Sonidos ambientales
  - `sfx/` - Efectos de sonido
- `fonts/` - Fuentes tipográficas
- `shaders/` - Shaders para efectos visuales
- `videos/` - Cutscenes y vídeos

## Formatos

- Sprites: PNG (Aseprite)
- Audio: OGG y MP3
- Fuentes: TTF o OTF

## Estilo Visual

### Pixel Art

- Resolución base: 320x180 píxeles
- Escala: x4 para pantalla completa (1280x720)
- Estilo: Pixel art detallado con colores vibrantes

### Sprites

- Jugador: 16x16 píxeles base
- NPCs: 16x16 píxeles base
- Vehículos: 32x32 píxeles base
- Tilesets: 16x16 píxeles por tile

### Animaciones

- Frame rate: 12 FPS para personajes
- Transiciones suaves entre frames

## Referencias Visuales

- Stardew Valley (estilo general)
- Celeste (animaciones de personajes)
